# cpp-hotel-booking-calculator

A C++ program that calculates hotel booking charges based on room cost, number of rooms, length of stay, discount rules, and sales tax.

## Features
- User input for room cost, rooms booked, days booked, and tax rate
- Tiered booking discounts based on number of rooms
- Additional discount for stays of 3 or more days
- Sales tax calculation
- Formatted billing output
